Setup Installation

Follow these step inorder to setup volant design system for your next awsome project

Step 1 : Clone from repository

Copy below line of code and paste on your command line

$git clone https://github.com/etsadhakal/volant-ds.git
Step 2 : Go to project

Go to your cloned directory

$cd volant-design-system
Step 3 : Run Npm Command

In order to run this command you need to have Node.js with higher than version Node.js 8.x

npm install

By installing this will install the necessary dependencies for to run the design system

npm run build

Use this command line and you see the build will be prepared.
